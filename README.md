# Easeword
基于MVVM开发的小项目

说明:
项目中使用到的字体文件未添加到仓库，因此若要在本地构建并运行项目，需要手动下载对应字体文件
杨任东竹石 字体文件对应目录文件 /app/src/main/assets/fonts/yrdzs.ttf
说文解字 字体文件对应目录文件 /app/src/main/assets/fonts/swjz.ttf
